TableViewViewModelable

Presenter protocol designed as data-sources and delegate for tableviews. The methods can be used for the designated places, or any other purposes.

  • numberOfRows(in:for:)

    This method designated to be called from tableView:numberOfRowsInSection:

  • numberOfSections(for:)

    This method designated to be called from numberOfSectionsInTableView:

    In default, returns 1.

  • cellForRow(at:for:)

    This method designated to be called from tableView:cellForRowAtIndexPath:

  • headerFor(section:for:) Default implementation

    This method designated to be called from tableView:viewForHeaderInSection:in delegate, when tableView:titleForHeaderInSection: in datasource returns nothing.

    In default, returns nil.

    Default Implementation

  • footerFor(section:for:) Default implementation

    This method designated to be called from tableView:viewForFooterInSection:in delegate, when tableView:titleForFooterInSection: in datasource returns nothing.

    In default, returns nil.

    Default Implementation

  • canEditRow(at:for:) Default implementation

    This method designated to be called from tableView:canEditRowAtIndexPath:

    In default, returns false.

    Default Implementation

  • canMoveRow(at:for:) Default implementation

    This method designated to be called from tableView:canMoveRowAtIndexPath:

    In default, returns false.

    Default Implementation

  • moveRowAt(from:to:for:) Default implementation

    This method designated to be called from tableView:moveRowAtIndexPath:toIndexPath:

    In default, does nothing.
